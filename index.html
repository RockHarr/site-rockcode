Definition of Done

  • Navegación por teclado + foco visible + contraste OK
  • LCP < 2.5s, CLS "Good"
  • Formularios: guardar/reanudar + PDF descargable
  • Búsqueda: sinónimos, ≤2 intentos

Accesibilidad primero

Componentes AA: botón, nav, modal con focus trap, tabs, alertas y formularios con etiquetas, descripciones y estados.

Rendimiento medible

Imágenes optimizadas, scripts defer, fuentes del sistema y pruebas Lighthouse (capturas antes/después).

Formularios que ayudan

Guardado en navegador, reanudación, validación clara y exportación a PDF con jsPDF.

Servicios RockCode

  • Auditoría AA / Performance para sitios públicos
  • Prototipos de trámites y formularios con PDF
  • Buscadores con sinónimos (FlexSearch / Lunr)
